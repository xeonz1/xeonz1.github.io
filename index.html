Horizon-variable nonlinear optimal control

2021/03 ~ 2021/06: Developed in XJTLU Robotics Lab

We propose a new type of horizon-variable indirect nonlinear optimal control algorithm. The algorithm uses quadratic programming to support box-limited input constraints and computes the solution efficienty with ricatti-recursion. The work is published in IEEE ICAC (International Conference on Automation and Computing).
